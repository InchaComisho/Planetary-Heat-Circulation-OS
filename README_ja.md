# 惑星熱・循環OS

## 地球温暖化・海洋熱含量・地球システム安定性を理解するための熱循環フレームワーク

**惑星熱・循環OS** は、地球温暖化を単なる「CO2排出による気温上昇」としてだけではなく、**蓄積された熱、弱体化した循環、劣化した炭素固定源、水循環の乱れ、生態系による冷却機能の低下**として理解するための概念フレームワークです。

この考え方は、CO2削減の重要性を否定するものではありません。
むしろ、排出削減は必要であるとした上で、それだけでは地球システムを時間内に安定化させるには不十分な可能性があると考えます。

なぜなら、すでに大量の熱が海に蓄積しており、同時にその熱を分散・吸収・固定・放出するための自然循環が弱まっている可能性があるからです。

簡単に言えば、こうなります。

> 地球温暖化とは、熱が地球に入りすぎているだけの問題ではない。
> その熱をうまく回し、吸収し、逃がすための地球の循環機能が弱っている問題でもある。

## 要旨

一般的な地球温暖化の説明では、温室効果ガスの排出によって放射強制力が高まり、地表気温が上昇すると説明されます。

これは重要な説明ですが、地球システム全体を見るには十分ではありません。

**惑星熱・循環OS** は、気候変動を次の二つの層から理解することを提案します。

1. **熱ストック**
   地球システムにすでに蓄積された余剰熱。特に海洋に蓄えられた熱。

2. **循環機能**
   海洋、大気、水循環、森林、土壌、微生物、植物プランクトン、生態系が、熱・水・炭素を分散し、緩衝し、固定し、放出する働き。

この視点に立つと、気候安定化には、将来の排出を減らすだけではなく、海洋熱含量、熱慣性、海洋循環、水循環回復、土壌・微生物再生、炭素固定源再生、生態系による冷却機能の回復を見る必要があります。

本稿は、完成された科学モデルではありません。
マスター / inchacomusho / InchaComisho による、公開仮説・概念フレームワーク・オープンリサーチ草案です。

## 1. なぜこのフレームワークが必要なのか

地球温暖化は、しばしば次のような単純な流れで説明されます。

```text id="f7jzd4"
CO2が増える
    -> 放射強制力が増える
    -> 地表気温が上がる
```

この説明は重要です。
しかし、それだけでは地球システム全体を説明しきれません。

地球は、静止した物体ではありません。
地球は、熱と循環によって動く巨大なシステムです。

地球は太陽からエネルギーを受け取り、海や陸に熱を蓄え、大気と海洋によって熱を運び、水を蒸発と降雨で循環させ、植物や微生物によって炭素を固定し、最終的にエネルギーを宇宙へ放出しています。

そのため、気候安定性を考えるには、次の問いが必要です。

* 熱はどこから入るのか
* 熱はどこに蓄積するのか
* 熱はどう移動するのか
* 海はどれだけ熱を蓄えているのか
* 水循環は地表温度をどう調整しているのか
* 森林、土壌、微生物、湿地、植物プランクトンは炭素固定にどう関わるのか
* 生態系の劣化は自然の冷却・緩衝機能をどれだけ弱めるのか
* 熱ストックが増え続ける一方で、循環が弱ると何が起きるのか

**惑星熱・循環OS** は、これらの問いを見えるようにするための枠組みです。

## 2. 中心定義

**惑星熱・循環OS** とは、地球を統合された熱循環システムとして見るための概念的なOSモデルです。

ここでいうOSとは、コンピューターのOSを比喩として用いたものです。

コンピューターのOSは、メモリ、処理、保存、通信を調整します。
同じように、地球システムは、熱、水、炭素、栄養、生命、エネルギーの流れによって安定しています。

惑星熱・循環OSは、地球温暖化を次の複合問題として扱います。

* 熱の蓄積
* 海洋への熱貯蔵
* 海洋循環と大気循環
* 水循環の乱れ
* 炭素循環の劣化
* 土壌・微生物循環の低下
* 生態系による冷却機能の喪失
* 気候ティッピングポイントのリスク
* 熱慣性による気候応答の遅れ

これらの流れが安定している間、地球は変化を緩衝できます。

しかし、これらの流れが弱まると、地球システムは不安定になります。

## 3. 重要な視点：余剰熱の多くは海に蓄積している

温暖化を人間が直接感じる場所は、大気です。

猛暑、熱波、夜の暑さ、干ばつ、豪雨。
これらは地表や大気で起こるため、私たちは「空気が暑くなっている」と感じます。

しかし、地球温暖化による余剰熱の大部分は、大気ではなく海に入っています。

海は空気よりもはるかに大きな熱容量を持っています。
そのため、海は膨大なエネルギーを吸収し、ゆっくりと温度を変えながら、熱を蓄え続けます。

これが、**海洋熱含量** が長期的な温暖化を理解する上で重要な理由です。

地表気温は年ごとに上下します。
しかし、海洋熱含量は、惑星全体のエネルギー不均衡をより深く示します。

もし地表気温が一時的に横ばいに見えても、海が熱を吸収し続けているなら、地球システム全体では熱が蓄積し続けていることになります。

## 4. 海洋熱含量とは何か

**海洋熱含量** とは、海洋にどれだけの熱エネルギーが蓄えられているかを示す指標です。

海のさまざまな場所と深さの温度を測定し、その熱量を積算することで求められます。

海洋熱含量が重要なのは、次の理由からです。

* 海は地球温暖化による余剰熱の大部分を吸収する
* 海洋熱は熱膨張による海面上昇に関わる
* 暖かい海は海洋熱波を強める
* 海洋熱は嵐、降雨、蒸発に影響する
* 海洋熱はサンゴ礁や海洋生態系に負荷をかける
* 深海を含む蓄熱は、長い応答時間と熱慣性を生む

このフレームワークでは、海洋熱含量を二次的な指標として扱いません。

海洋熱含量は、**惑星の熱ストックを示す中心指標**です。

## 5. 地球温暖化を「熱ストック」と「循環弱化」で見る

惑星熱・循環OSでは、地球温暖化を二つの問題に分けて見ます。

### 5.1 熱ストック

熱ストックとは、すでに地球システムに蓄積された余剰熱のことです。

これには次のものが含まれます。

* 海洋熱含量
* 陸域の蓄熱
* 大気中の熱
* 雪氷圏のエネルギー不均衡
* 深海層に蓄積された熱
* 蓄積熱の遅れた放出や再分配

排出削減は、将来の熱入力を減らすことができます。
しかし、すでに海に蓄積した熱をすぐに消すわけではありません。

### 5.2 循環弱化

循環弱化とは、自然システムが熱・水・炭素を移動し、緩衝し、吸収し、調整する能力が低下することです。

これには次のものが含まれます。

* 海洋成層化
* 鉛直混合の低下
* 海流の変化
* 降雨パターンの乱れ
* 土壌水分の低下
* 森林喪失と植生劣化
* 微生物循環の劣化
* 湿地喪失
* 炭素固定能力の低下
* 生態系ストレス

熱ストックが増え、同時に循環が弱まると、問題は単なる平均気温上昇ではなくなります。

それは、地球システム全体の安定性低下になります。

## 6. なぜCO2単独フレームでは不十分なのか

CO2削減は必要です。

このフレームワークは、温室効果ガスの役割を否定するものではありません。
排出が重要でないと主張するものでもありません。

しかし、CO2だけに焦点を当てると、二つの大きな問題を見落としやすくなります。

### 6.1 すでに蓄積した熱の問題

排出量が減少しても、海に蓄えられた熱はすぐには消えません。

海には熱慣性があります。
つまり、気候システムの応答には遅れがあります。

蓄積された海洋熱は、次のような形で影響し続けます。

* 海面上昇
* 海洋熱波
* サンゴ白化
* 極端降雨
* 熱帯低気圧
* 氷の融解
* 生態系ストレス
* 長期的な気候フィードバック

したがって、気候安定化には、将来の排出だけでなく、すでに蓄積された熱を見る必要があります。

### 6.2 自然冷却・循環機能の問題

地球には、本来の冷却・緩衝システムがあります。

たとえば次のようなものです。

* 蒸発
* 雲
* 降雨
* 海洋循環
* 森林の蒸散
* 土壌の保水
* 微生物分解
* 植物とプランクトンによる炭素固定
* 湿地とバイオマスによる蓄積
* 植生による地表冷却

もしこれらが弱まれば、地球は熱を緩衝し、気候を安定させる能力の一部を失います。

この意味で、地球温暖化は排出問題であると同時に、循環機能の問題でもあります。

## 7. 人間の体にたとえるとわかりやすい

惑星熱・循環OSは、人間の体にたとえると理解しやすくなります。

人間の体が過熱したとき、必要なのは熱源を減らすことだけではありません。

体には次の機能も必要です。

* 血流
* 発汗
* 水分補給
* 臓器保護
* 体温調節
* 回復時間

もし血流や発汗の仕組みが働かなければ、少しの熱でも危険になります。

地球システムも似ています。

CO2削減は、熱源を減らすことに似ています。
しかし、地球には循環機能も必要です。

* 海洋循環
* 大気循環
* 水循環
* 森林蒸散
* 土壌水分
* 微生物循環
* 生態系による緩衝
* 炭素固定

これらが弱まれば、地球は熱を調整しにくくなります。

## 8. 気候ティッピングポイントとの関係

気候リスクは、気温が少しずつ上がることだけではありません。

大きな懸念は、地球システムの一部が閾値を超え、変化が自己強化的、急激、または戻りにくい状態になることです。

このような閾値は、**気候ティッピングポイント** と呼ばれます。

気候科学で議論される例には、次のようなものがあります。

* 温暖な海域のサンゴ礁衰退
* 氷床の不安定化
* 永久凍土融解
* 熱帯雨林の枯死
* 主要な海洋循環の変化

惑星熱・循環OSでは、ティッピングポイントを惑星OSの機能障害または動作モードの変化として見ます。

たとえばサンゴ礁のティッピングポイントは、単なる生物学的な出来事ではありません。
海洋熱、生態系ストレス、気候強制力が、生きた循環システムを安定範囲の外へ押し出したサインでもあります。

だからこそ、熱ストックと循環機能を同時に監視する必要があります。

## 9. 気候対策の三層構造

このフレームワークから見ると、気候対策には三つの層が必要です。

### 第1層：将来の熱入力を減らす

これには次のようなものが含まれます。

* 温室効果ガス排出削減
* 再生可能エネルギーへの移行
* エネルギー効率化
* 化石燃料依存の低減
* 産業構造の再設計
* 廃棄と過剰消費の削減

この層は必要です。
将来の熱圧力を小さくするためのブレーキです。

### 第2層：すでに蓄積した熱に対応する

この層では、すでに蓄積された熱への曝露をどう下げるかを考えます。

これには次のようなものが含まれます。

* 海洋熱含量の監視
* 都市熱の低減
* 地表冷却戦略
* 水循環回復
* 熱リスクマッピング
* 海洋熱波への対応
* 極端な熱ストレスの緩和

ここで、**地球直接冷却** の概念が関係します。

地球直接冷却は、排出削減の代替として理解すべきではありません。
蓄積熱と弱体化した冷却機能に対応するための、補完的な層として提案されます。

### 第3層：循環機能を回復する

これには次のようなものが含まれます。

* 森林再生
* 土壌再生
* 微生物循環の回復
* 湿地再生
* 保水力の回復
* 蒸発散の回復
* 海洋循環補完
* 植物プランクトンと海洋生態系の回復
* 炭素固定源の再生

この層は、地球本来の冷却・緩衝・炭素固定機能を回復することを目的とします。

## 10. 地球直接冷却との関係

惑星熱・循環OSは、地球直接冷却を考えるための概念的基盤です。

地球直接冷却とは、単に人工的に地球を冷やす試みではありません。

このフレームワークでは、次のように定義します。

> 地球直接冷却とは、地球本来の自然冷却機能と循環機能を補完・回復するための提案である。

この考え方には、次のような方向性が含まれます。

* 海洋循環補完
* 地表熱の低減
* 水循環回復
* 土壌と植生の再生
* 都市冷却
* 生態系による冷却
* 炭素固定源再生
* 海洋熱含量と熱リスク領域の監視

目的は、排出削減を否定することではありません。

目的は、欠けている層を追加することです。

```text id="48d1st"
排出削減
+ 熱ストックの認識
+ 循環機能の回復
+ 生態系再生
+ 自然補完型の直接冷却
```

## 12. このフレームワークが変える見方

従来の単純化された見方では、次のように考えがちです。

> CO2排出を減らせば、いずれ温暖化は止まる。

惑星熱・循環OSでは、次のように考えます。

> CO2排出削減は必要である。
> しかし地球システムには、蓄積された海洋熱、熱慣性、弱った水循環、劣化した土壌、ストレスを受けた生態系、弱体化した炭素固定源が存在する。
> 気候安定化には、これらを同時に扱う必要がある。

これは気候科学の否定ではありません。

単一の入力変数だけでなく、地球システム全体の安定性へ視野を広げる試みです。

* 植物プランクトン
* 生態系

この視点に立つと、地球温暖化は単なるCO2問題ではありません。

それは次のような複合問題です。

* 蓄積された海洋熱
* 熱慣性
* 海洋・大気循環の弱化
* 水循環の乱れ
* 土壌・微生物システムの劣化
* 炭素固定能力の低下
* 生態系ストレス
* 気候ティッピングポイントへの接近

したがって、包括的な気候対策には次の三つが必要です。

1. 排出削減
2. 熱ストックの監視と対応
3. 惑星循環機能の回復

これが、惑星熱・循環OSの目的です。

地球全体のスケールで気候安定化を考えるための見取り図です。
